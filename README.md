## 快速上手

1. 克隆仓库  

   ```bash
   git clone https://github.com/ttpoker/texas-cash-game-platform.git
   cd texas-cash-game-platform
   ```

2. 安装依赖（按技术栈）  

   ```bash
    sh build  
   ```
3. 配置文件  

   ```bash
   cp config/config.example.yml config/config.yml
   ```  
   编辑 `config.yml`，配置数据库、日志、端口。

4. 启动服务端  

   ```bash
   npm run start
   ```

5. 多端接入  
   - 配置服务端 URL 到 Web / APP / 小程序，通过 API / WebSocket 接入。

## ❓ FAQ / 常见问题

**Q: 现金桌支持多少玩家同时在线？**  
A: 单桌最多9人，支持无限开桌，服务器架构支持横向扩展。

**Q: 抽水比例可以自定义吗？**  
A: 可以。所有现金桌参数（抽水%、封顶、时间银行等）均可在后台配置。

**Q: 源码是什么技术栈？**  
A: 【后端：C++，后台：Java ，数据库：MySQL，前端：Unity】

**Q: 是否支持多币种 / 真实支付？**  
A: 已集成代币系统，可替换为 USD、THB、VND 等真实支付通道。
